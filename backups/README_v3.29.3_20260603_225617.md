# 深渊文字迷宫 (Abyss Text Labyrinth)

## 项目简介

HTML5 文字迷宫肉鸽游戏，复古终端风格界面，浏览器直接运行。

**路径**: `/Users/xiaoyu/Desktop/Rogue/`
**运行**: 浏览器打开 `abyss-labyrinth.html`

## 游戏特性

### 核心玩法
- **🏰 城堡大厅**：游戏起始层，含 💚治疗师 / 🏪装备商人 / 🧙老贤者 / 🏗️铁匠 NPC，火炬🔥石柱🏛️王座👑红毯◆装饰，走进 🌀 传送门开始冒险。开始界面城堡主题深紫星空背景+Canvas城堡剪影
- 🏰 **回城传送门**：地城 ~15% 概率出现，返回城堡补给后再回到原楼层
- ✅ **随机漫步法** 生成连通迷宫，27×15
- **碰撞战斗**：移动到敌人坐标触发，支持自动/手动两种模式
- **⭐精英怪**：~25% 概率出现，👑 皇冠图标+金色脉冲，2×HP/1.5×ATK，10 种随机能力，击败掉装备
- **🐲Boss 战**：每 5 层，单回合制，击败后 🌀 传送门才出现，随机掉落高品质装备（品质≥蓝，属性×1.4，词缀×1.5）
- **永久死亡**：HP 归零显示存活层数，可重新开始，带 💀 死亡动画（闪红+震颤+粒子爆炸）
- **🧙 随机事件 NPC**：地城 ~20% 刷出，50/50 好运/倒霉，35% 触发戏剧性特殊事件，正面全屏烟花🎆+升级音效 / 负面骷髅💀+受伤音效
- **🏆 成就系统**：14个里程碑成就，localStorage 持久化，按 P 键查看
- **🏚️ 特殊房间**：~15% 概率出现 8 种特殊房间：💰宝藏室 / ⚔️竞技场 / ⛓️祭坛 / 📚图书馆 / 🎰赌博商人 / 🕳️时空裂缝 / 🏋️训练场 / 🌟许愿井
- **🔥 环境效果**：~20% 概率 ☠️毒雾（每步掉血）/ 🌑黑暗（视野缩小）/ 🌀魔力紊乱（技能CD翻倍）
- **🧪 铁匠铺**：城堡 🏗️ 铁匠处 🔨强化装备（+1/+2消费金币）/ ⚗️合成装备（2件合1，概率升级品质）

### 战斗系统
- **10 种战斗机制**：穿透（真实伤害） / 双连击 / 减伤 / 护盾 / 冰冻 / 眩晕 / 点燃 / 脆弱 / 暴击 / 生命偷取
- **状态效果**：☠️中毒(5HP/回合) / 🩸流血(5%HP/回合) / 🔥点燃(3回合) / 💔脆弱(承伤+20%)
- **5 种战斗特效**：🔥火球 ⚡闪电 🌪️飓风 ⚔️刀剑 👊铁拳
- **自动攻击开关**：右上角 ⚔️/✋ 切换，默认自动

### 角色系统
- **4 职业**：🤺战士(高防暴击) 🔮法师(穿透/药水) 🥷盗贼(闪避/折扣) 🧝牧师(治疗/抗性)
- **18 天赋**：⚔️战斗 🧭探索 ❤️生存 三系+Q键主动技能
- **升级弹窗**：每次升级随机 3 选 1，16 种选项涵盖所有战斗属性，竖排详情一一对应

### 装备系统
- **9 槽位**：⚔️主手 🛡️副手 🎩头盔 👕胸甲 🧤护手 👢靴子 💍戒指Ⅰ 💍戒指Ⅱ 🧿项链
- **9 档品质**：破烂→神话，品质着色+词缀数递增
- **词缀系统**：4 类 20+ 条词缀（含穿透/连击/冰冻/眩晕/点燃/脆弱等新属性）
- **9 套套装**：2/4/6 件触发效果
- **装备对比弹窗**：获得装备时弹出新 vs 旧对比，手动选择替换或放弃
- **💀 诅咒装备**（v3.23.0）：6种诅咒，属性极强+副作用，不可替换，城堡净化师解除
- **🧪 药水系统**（v3.26.0）：11种药水（生命/法力/护盾/速度/力量/毒药/贪婪诱饵/时空凝滞/狂战士之血/荆棘反伤/炼金盲盒），4格背包，1-4快捷键

### 物品与商店
- **宝箱**：100% 掉落装备（普通层 45%/Boss 层 80%/商店层 30%）
- **药水**：~50% 概率出现，1-2 瓶/层
- **商店**：每 3 层 🏪 杂货商人，6 随机装备+3 药水

### 排行榜（v3.18.0）
- **评分公式**：`(层数×100 + 击杀×10 + 金币×1) × 职业系数`
- **职业系数**：战士1.0 / 法师1.1 / 盗贼1.2 / 牧师1.0
- **存储**：localStorage 本地排行，前 100 名
- **入口**：主菜单 🏆 按钮 + 死亡结算界面

### 工匠工坊（v3.22.0）
- **永久升级**：死亡后点数保留，12项升级树
- **点数获取**：`floor×2 + kills/2`（死亡结算）
- **升级项**：🔨HP ⚔️攻击 🛡️防御 🔰护盾 💥暴击 🩸吸血 🍀幸运 💎金币 🪙折扣 ⭐天赋 🔄再生 👟速度
- **入口**：主界面 ⚒️ 工坊按钮
- **展示**：排名/玩家/职业/层数/击杀/得分/日期，前三金银铜高亮

## 操作方式

| 按键 | 功能 |
|------|------|
| ↑↓←→ / WASD | 移动角色 |
| C | 角色属性面板（装备/天赋/战斗属性全览）|
| Tab | 装备详情面板 |
| Q | 主动技能（需先学习天赋）|
| ESC | 关闭面板/商店 |
| ` | 切换精灵/文字模式 |
| 右上角 ⚔️ | 自动攻击开关 |

调试键：`0` 无敌 / `9` 跳Boss层 / `8` 跳商店层 / `7` +500金币 / `6` 满血 / `空格` 下层
