## 一、游戏概述

### 世界观
在时间尽头，有一座被遗忘的深渊迷宫。它吞噬了无数冒险者的灵魂——而你，是下一个挑战者。

### 核心目标
探索随机生成的楼层迷宫，击败怪物、收集装备、学习天赋，存活到最深层。

### 游戏类型
**HTML5 文字迷宫 Roguelike**：回合制移动 + 碰撞战斗 + 永久死亡 + 元进度（工匠工坊）。

## 三、核心系统

### 3.1 地图与探索

- **随机漫步法**生成 27×15 连通迷宫
- **5 大主题**随层数切换：🌑深渊(1-5) → 🌲森林(6-10) → 🕳️洞穴(11-15) → 🌋火山(16-20) → ❄️冰原(21-25)
- **每层元素**：墙壁/地板/敌人/宝箱/药水/出口/传送门
- **城堡大厅**（第0层）：起始层，含 5 个功能 NPC + 🌀 传送门进入地城
- **回城传送门**：地城 ~15% 概率出现，返回城堡补给后再回到原楼层

### 3.2 战斗系统

**基础公式**：`伤害 = 攻击力 - 防御力（最低1点）`

**10 种战斗机制**：

| 机制 | 说明 |
|------|------|
| 🗡️ 穿透 | 无视防御的真实伤害 |
| ⚔️ 连击 | 概率再攻击一次 |
| 🛡️ 减伤 | 受到的伤害降低 X% |
| 🔰 护盾 | 战斗开始额外生命值，优先消耗 |
| ❄️ 冰冻 | 敌人跳过 1 回合 |
| 😵 眩晕 | 敌人跳过 1 回合 |
| 🔥 点燃 | 每回合掉血，持续 3 回合 |
| 💔 脆弱 | 敌人承伤 +20%，持续 2 回合 |
| 💥 暴击 | 1.5× 伤害（天赋可提升至 2.0×） |
| 🩸 生命偷取 | 击杀敌人恢复 HP |

**自动/手动切换**：右上角 ⚔️/✋ 按钮，默认自动攻击。

### 3.5 Roguelike 机制

- **永久死亡**：HP 归零 → 死亡动画（闪红+震颤+粒子爆炸）→ 结算界面 → 回到开始
- **元进度**：工匠工坊的永久升级在死亡后保留
- **存档系统**：自动存档（楼层过渡/击杀后/传送门双向），继续游戏从存档恢复
- **排行榜**：localStorage 本地排行，评分公式 `(层数×100 + 击杀×10 + 金币×1) × 职业系数`

## 十二、常见问题

**Q: 为什么我的药水没放进背包？**
A: 踩到药水后会弹出选择框，需要点击「放入背包」才会存储。直接走过不会自动拾取。

**Q: 护盾为什么被打穿了？**
A: 护盾是额外HP，在受到伤害时优先消耗。如果单次伤害超过护盾值，超出部分会扣HP。

**Q: 诅咒装备无法替换怎么办？**
A: 诅咒装备被锁定，必须回城堡找 ✨净化师（50金币）解除后才能替换。

**Q: 时空裂缝的门点不了？**
A: 需要先击败守卫裂缝的怪物（1-2只），然后才能点击 🔴红门或 🔵蓝门。

**Q: 训练场打不过幻影？**
A: 训练场不会真正死亡，HP归零会自动恢复满血。可以反复进入直到击败。

**Q: 为什么我按 1 没有秒杀？**
A: 秒杀快捷键从 v3.26.0 起改为 **5**。1-4 现在是药水背包键。

**Q: 精灵模式是什么？**
A: 按 ` 键切换，用像素精灵图替代文字显示地图元素（墙壁/地板/玩家/怪物/Boss等）。
