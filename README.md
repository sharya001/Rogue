# 深渊文字迷宫 (Abyss Text Labyrinth)

HTML5 文字迷宫肉鸽游戏，复古终端风格，浏览器直接运行。

**运行**: 浏览器打开 `abyss-labyrinth.html`

## 文件结构

```
Rogue/
├── abyss-labyrinth.html    # 构建产物（python3 build.py）
├── build.py                # 构建脚本 + 版本同步
├── VERSION                 # 版本号
├── CHANGELOG.md            # 版本更新日志
├── GUIDE.md                # 玩家指南
├── DEVELOPMENT.md          # 开发规范
├── AGENTS.md               # Agent 专用指令
├── src/                    # 源代码
│   ├── css/                # 11 个 CSS 文件
│   ├── html/               # 2 个 HTML 片段
│   ├── js/                 # 17 个 JS 模块
│   └── templates/          # 3 个 HTML 模板
├── backups/                # 版本备份
└── temp/                   # 临时文件
```
